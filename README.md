# Pong Delphi

A simple implementation of the classic Pong game, written entirely in Pascal using Delphi.

## Overview

This repository contains a full source code project for Pong, developed with the Delphi environment. The project is a great starting point for anyone interested in game development or learning how to work with graphical applications in Pascal.

**Features:**
- Classic Pong gameplay: paddle controls and ball physics
- Single-player mode against computer AI
- Adjustable game speed and difficulty
- Clean, readable code ideal for learning

## Getting Started

### Prerequisites

- [Delphi IDE](https://www.embarcadero.com/products/delphi) (any modern version should work)
- Basic understanding of Pascal programming language

### Setup & Running

1. **Clone this repository:**
   ```bash
   git clone https://github.com/NickHirras/pong-delphi.git
   ```
2. **Open the project file in Delphi.**
3. **Build and run the project** via the IDE's "Run" or "Build and Run" options.

### Controls

- **Player 1:** Use `Up` and `Down` arrows to move the paddle.
- **Player 2 / AI:** Uses automatic movement (if enabled).
